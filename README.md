# CMS
Web application written with usage of Java and Spring framework. Content Management System with CRUD functionalities.


# Technologies : 

- Java 1.8

- Spring
 Boot, Spring Security
- Hibernate, JPA


- Thymeleaf
- Angular 4, Bootstrap
- JUnit
- Mockito
# Features : 

- user login

- user registration
- posts displaying on main page

After signing in : 
- post adding

- post editing

- post removing
